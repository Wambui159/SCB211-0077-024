Leah Wambui Portfolio Website

This is my personal portfolio website built using HTML, CSS, and JavaScript. The site showcases my skills as a web developer and my passion for cooking through two of my favorite recipes: Pizza and Burger.

Project Structure

portfolio-website/ 
├── css/ 
│ └── style.css 
├── images/ 
│ ├── hero.jpg 
│ ├── profile.jpg 
│ ├── pizza.jpg 
│ └── burger.jpg 
├── js/ 
│ └── script.js 
├── index.html 
├── about.html 
├── pizza.html 
└── burger.html
|____READMe

-index.html: The landing page with a hero section, introduction, and navigation.
-about.html: The page that provides detailed information about me, my skills, and fun personal facts.
-pizza.html: A recipe page featuring a homemade pizza recipe with ingredients, step-by-step instructions, and nutritional information.
-burger.html: A recipe page featuring a classic juicy burger with similar structure.
-css/style.css: Contains the styling for all pages to ensure a consistent look and responsive design.
-js/script.js: Contains basic JavaScript (currently logging a message to the console on page load) to provide a foundation for future interactive features.
- images/: Folder containing all the image assets used in the website.

How to Navigate the Website

1. Landing Page (Home):
   - Features a hero section with a background image, a welcome message, and a call-to-action button leading to the About page.
   - A navigation menu is available at the top to access all the pages.

2.About Page:
   - Contains a profile photo, a brief biography, a list of my technical skills, and fun facts about me.

3.Pizza Recipe Page:
   - Provides a detailed recipe for a delicious homemade pizza including ingredients and step-by-step cooking instructions.

4.Burger Recipe Page:
   - Offers a detailed recipe for a classic juicy burger, including ingredients and step-by-step instructions.

Technologies Used

-HTML: To structure the content of the website.
- CSS: For styling and layout, ensuring the site is visually appealing and responsive.
- JavaScript: To add interactivity to the website (e.g., logging a message when the site loads and potential future enhancements).

How to Run the Project

1. Download or clone the repository to your local machine.
2. Open the `index.html` file in your preferred web browser.
3. Use the navigation links to move between the Home, About, Pizza Recipe, and Burger Recipe pages.

Additional Information

- Image Sizing: You can adjust the size of the images using CSS (for example, by setting a `max-width` on the image tags) to ensure they display well on all devices.
- Responsive Design: The CSS includes media queries to ensure the website looks great on both desktop and mobile devices.
- Code Validation: HTML and CSS have been validated using the [W3C Validators](https://validator.w3.org/).
